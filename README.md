<h2>Content</h2>
<p>The website contains</p>
<ul>
  <li>Our Projects</li>
  <li>Project Documentaton</li>
  <li>Contact Information</li>
  <li>An About Us Page</li>
  <li>Community Hangout</li>
</ul>

<h2>Organizational Structure</h2>
<ul>
  <li>Homepage: This page contains links to the project pages</li>
  <li>Project Pages: These pages contain links to a project's Github Repositary, Privacy Policy, ToS, and whatever else may be relevant to that project</li>
  <li>Footer: The footer contains contact information, community hangout pages, and our brand philosophy</li>
</ul>

<h2>Tech Stack</h2>
<ol>
  <li>Hyper Text Markup Language (HTML): Used for the main structure of the website</li>
  <li>Cascading Style Sheets (CSS): Used for styling effects</li>
  <li>JavaScript (JS): Used for interactive actions such as a copy button</li>
</ol>

> [!NOTE]
> Since this is a static website, we do not use server-side languages in our stack such as C# or Python.
