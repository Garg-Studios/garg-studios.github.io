Exchange Bot: Privacy Policy 

This is the Privacy Policy for Exchange Bot. By using Exchange Bot, you agree and understand the terms listed below


1. Data Collection and Processing

Exchange Bot operates primarily as a stateless utility. To fulfill user requests, we temporarily process command arguments (such as stock tickers, fiat currency codes, and cryptocurrency symbols) provided during interactions with the bot. We do not permanently store or log private user text, user profiles, or server message histories.


2. Third-Party Data Transmission

To provide real-time market insights, the financial tickers you input are transmitted to the following third-party application programming interfaces (APIs):

  • Yahoo Finance API (Query1): For equity and stock market data.
  • Kraken API: For cryptocurrency pricing and market data.
  • Frankfurter API (frankfurter.dev): For foreign fiat currency exchange rates.

No Personally Identifiable Information (PII), Discord usernames, or unique Discord IDs are transmitted to these third parties.


3. Data Security

Because Exchange Bot does not retain user databases or personal history, your data cannot be exposed or sold. Any operational data processed in memory during a command is discarded immediately upon displaying the results to the Discord channel.


4. Changes to This Policy

If Exchange Bot introduces features requiring a permanent database (such as personalized asset tracking or watchlists), this policy will be updated, and explicit user consent will be requested within the application prior to storing user data.
