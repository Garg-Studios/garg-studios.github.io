Exchange Bot Documentation

This page showcases all commands built into Exchange Bot

Core Commands

  • Check a stock ticker: /checkstock
  • Check a cryptocurrency: /checkcrypto
  • Check a currency exchange rate: /convertcurrency
  • Contact Garg Studios about an issue: /help
  • Check Exchange Bot Updates: /update
  • Check if the bot is online with a simple /ping

/checkstock

Parameters:

  • Ticker: this is the stock symbol which will be checked (e.g., GOOG, APPL, AMZN)

Returns:

  • Company Name
  • Stock Price
  • Exchange
  • 52-Week Range
  • 1-Day Range

This command queries the Yahoo Finance API at query1.finance.yahoo.com/v8/finance/chart/{TICKER}

Inputted Text: /checkstock ticker: AAPL

Inputted Text: /checkstock ticker: RANDD

This error is caused by a particular stock symbol not being found by the API

/checkcrypto

Parameters:

  • Crypto: This is the Crypto Symbol (e.g., BTC, ETH, SOL)

Returns:

  • Price

This command queries the Kraken API at api.kraken.com/0/public/Ticker?pair={api_ticker}USD. However, before this, the ticker is modified by the program to make it readable by the Kraken API. Check the GitHub Repo for more information.

Inputted Text: /checkcrypto crypto: BTC

Errors

Inputted Text: /checkcrypto crypto: ASDFE

This error is caused by a particular crypto symbol not being found in Kraken

/convertcurrency

Parameters:

  • Currency 1: This is the currency you are converting from
  • Amount: This is the amount of money you are converting
  • Currency 2: This is the currency you are converting to

Returns:

  • Your Specified Currency
  • British Pound
  • Euro
  • American Dollar
  • Japanese Yen
  • Indian Rupee
  • The date that the exchange rates were last updated on

This command queries the Frankfurter API at api.frankfurter.dev/v1/latest?base={currency}

Inputted Text: /convertcurrency currency1: USD currencyamount: 1 currency2: CNY

Errors

Inputted Text: /convertcurrency currency1: RRTR currencyamount: 1 currency2: USD

This error happens when the first or both currency parameters are invalid. Since the API requires the currency to be embedded, an invalid first currency will cause this error.

Inputted Text: /convertcurrency currency1: USD currencyamount: 1 currency2: RIRR

This error happens when the first currency is valid but the second is not. This error indicates the API was able to return a JSON file, but it could not parse the JSON file with the key provided.
